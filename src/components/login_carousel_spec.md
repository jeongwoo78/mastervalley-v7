## 4. 회전 로직

### 1회차 (첫인상)
원본 → 18작품, 정해진 순서대로 느리게→빠르게 가속.
사조 → 거장 → 동양화 순서로 2800년 미술사 펼침.

### 2회차~ (무한 반복)
- 원본은 항상 첫 번째 (1.2초)
- 나머지 17장은 **랜덤 셔플**
- 같은 가속 패턴 적용
- 매 회차 순서가 달라져서 "이번엔 뭐가 나올까" 효과

## 5. 코드 변경 사항 (LoginScreen.jsx v77)

### 변경 포인트
1. `CAROUSEL_SLIDES` → `SLIDES_BY_LANG` (언어별 18장 config)
2. `LABELS` 객체 (아티스트 라벨)
3. `TIMINGS` 배열 (가속 타이밍 18개)
4. `shuffleArray()` 함수 (2회차부터 셔플)
5. `buildSlides(lang)` 함수 (언어별 슬라이드 빌드)
6. `useEffect` — 가속 + 셔플 로직
7. `slideOrder` state — 현재 순서 관리
8. 페이드 CSS: `transition: opacity 1s` → `0.25s`

### 이미지 경로 규칙
```
/login/{lang}/{artist}.jpg
```
예: `/login/ko/vangogh.jpg`, `/login/fr/klimt.jpg`
